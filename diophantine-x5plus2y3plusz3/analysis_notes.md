# Analysis Notes: $x^5 + 2y^3 + z^3 = 0$

## Problem Statement

Find all integer solutions $(x, y, z) \in \mathbb{Z}^3$ to

$$x^5 + 2y^3 + z^3 = 0$$

or prove the non-existence of integer solutions.

---

## Main Result

The equation has **infinitely many integer solutions**, including two explicit infinite parametric families:

$$\text{Family 1:} \quad (x, y, z) = (-n^3,\; n^5,\; -n^5), \qquad n \in \mathbb{Z}.$$

$$\text{Family 2:} \quad (x, y, z) = (-n^3,\; 0,\; n^5), \qquad n \in \mathbb{Z}.$$

**Verification of Family 1:**

$$(-n^3)^5 + 2(n^5)^3 + (-n^5)^3 = -n^{15} + 2n^{15} - n^{15} = 0. \checkmark$$

**Verification of Family 2:**

$$(-n^3)^5 + 2 \cdot 0^3 + (n^5)^3 = -n^{15} + 0 + n^{15} = 0. \checkmark$$

---

## Weighted Homogeneity

Assign weights $\text{wt}(x) = 3$, $\text{wt}(y) = 5$, $\text{wt}(z) = 5$. Then each term has weighted degree 15:

- $x^5$: weighted degree $5 \times 3 = 15$,
- $2y^3$: weighted degree $3 \times 5 = 15$,
- $z^3$: weighted degree $3 \times 5 = 15$.

**Consequence:** If $(a, b, c)$ is any integer solution, then $(t^3 a, t^5 b, t^5 c)$ is also an integer solution for every $t \in \mathbb{Z}$, since

$$
(t^3 a)^5 + 2(t^5 b)^3 + (t^5 c)^3
= t^{15}(a^5 + 2b^3 + c^3) = t^{15} \cdot 0 = 0.
$$

Thus it suffices to find a single **seed** non-trivial solution in order to generate an infinite parametric family.

---

## Derivation of the Parametric Families

### Step 1 — Reduction by setting $z = -y$

Setting $z = -y$, the equation becomes

$$x^5 + 2y^3 + (-y)^3 = x^5 + 2y^3 - y^3 = x^5 + y^3 = 0.$$

So we need $x^5 = -y^3$.

**Seed:** Take $x = -1$, $y = 1$: $(-1)^5 = -1$ and $-1^3 = -1$. ✓

By weighted homogeneity (weights $\text{wt}(x) = 3$, $\text{wt}(y) = 5$ in the reduced equation $x^5 + y^3 = 0$), the full family is:

$$x = -t^3, \quad y = t^5, \quad z = -t^5, \qquad t \in \mathbb{Z}.$$

**Direct verification:**

$$(-t^3)^5 + 2(t^5)^3 + (-t^5)^3 = -t^{15} + 2t^{15} - t^{15} = 0. \quad \checkmark$$

### Step 2 — Reduction by setting $y = 0$

Setting $y = 0$, the equation becomes

$$x^5 + z^3 = 0,$$

which requires $x^5 = -z^3$.

**Seed:** Take $x = -1$, $z = 1$: $(-1)^5 = -1 = -(1^3)$. ✓

By weighted homogeneity (reduced equation $x^5 + z^3 = 0$), the full family is:

$$x = -t^3, \quad y = 0, \quad z = t^5, \qquad t \in \mathbb{Z}.$$

**Direct verification:**

$$(-t^3)^5 + 2 \cdot 0^3 + (t^5)^3 = -t^{15} + 0 + t^{15} = 0. \quad \checkmark$$

### Step 3 — Generalisation via $z = \lambda y$ (setting $y = k z$)

Setting $z = \lambda y$ for a fixed $\lambda \in \mathbb{Z}$, the equation becomes

$$x^5 + (2 + \lambda^3) y^3 = 0.$$

This has solutions whenever $\gcd(5, 3) = 1$ and $2 + \lambda^3$ has a sign. Specifically, take $x = -(2+\lambda^3)^3 t^3$, $y = (2+\lambda^3)^5 t^5$, but the arithmetic simplifies when $\lambda = -1$ (Family 1, above) or when $2 + \lambda^3 = 0$, i.e., $\lambda = -\sqrt[3]{2}$ (not an integer). The cleanest special cases are $\lambda \in \{0, -1\}$, corresponding to Families 2 and 1.

## Infinitude of Solutions

**Claim:** The solution set is infinite.

**Proof:** Consider the restriction of Family 1 to $n \in \mathbb{N}$:

$$\varphi : \mathbb{N} \to \mathbb{Z}^3, \quad \varphi(n) = (-n^3, n^5, -n^5).$$

The map $n \mapsto n^3$ is strictly increasing on $\mathbb{N}$ (since $n^3 < m^3$ whenever $n < m$). Hence $\varphi$ is injective: if $\varphi(n) = \varphi(m)$ then $n^3 = m^3$, so $n = m$. An injective map from the infinite set $\mathbb{N}$ into the solution set certifies that the solution set is infinite.

---

## Connection to the Equation $x^5 + y^3 = 0$

Family 1 is generated by the sub-problem $x^5 + y^3 = 0$ (set $z = -y$). Over the integers, this equation is solved precisely by

$$x = -t^3, \quad y = t^5, \qquad t \in \mathbb{Z}$$

(up to sign: also $(t^3, -t^5)$ for any $t$), because $x^5/y^3 = -1$ requires the ratio to be $-(−1)^{15}$, and the minimal solution is $(x, y) = (-1, 1)$. By unique factorization in $\mathbb{Z}$, any solution must be of this form (or a root-scaled variant). The weighted-homogeneity argument then produces the full parametric family.

## Additional Parametric Families

A computational search over $|x|, |y|, |z| \leq 50$ reveals three further seed solutions
not accounted for by Families 1 and 2:

| Seed $(a, b, c)$ | Verification | Generated family $(a \cdot n^3,\; b \cdot n^5,\; c \cdot n^5)$ |
|---|---|---|
| $(-4, 8, 0)$ | $-1024 + 1024 + 0 = 0$ ✓ | $(-4n^3,\; 8n^5,\; 0)$ |
| $(-9, 27, 27)$ | $-59049 + 39366 + 19683 = 0$ ✓ | $(-9n^3,\; 27n^5,\; 27n^5)$ |
| $(-5, -5, 15)$ | $-3125 - 250 + 3375 = 0$ ✓ | $(-5n^3,\; -5n^5,\; 15n^5)$ |

Each negative seed (positive $x$) is obtained by the symmetry $(x,y,z)\mapsto(-x,-y,-z)$.

**Family 3:** $(x, y, z) = (-4n^3, 8n^5, 0)$:
$$(-4n^3)^5 + 2(8n^5)^3 + 0^3 = -4^5 n^{15} + 2 \cdot 8^3 n^{15} = (-1024 + 1024)\,n^{15} = 0.$$

All three extra families arise from seeds $(a, b, c)$ with $a^5 + 2b^3 + c^3 = 0$ that are
not in the span of the primary seeds, and each generates an infinite parametric family via
the weighted-homogeneity scaling $(a, b, c) \mapsto (a n^3, b n^5, c n^5)$.

---

## No Elementary Modular Obstruction

Unlike many Diophantine equations of this type, the equation $x^5 + 2y^3 + z^3 = 0$ is **everywhere locally solvable** — indeed, it has the explicit global solution Family 1, so no congruence obstruction can exist.
